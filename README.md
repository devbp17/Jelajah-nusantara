# Jelajah Nusantara
Aplikasi untuk berbagi tempat destinasi yang bisa dilihat oleh semua orang

# Fitur
- Login & Register
- Menambahkan gambar serta deskripsi
- Menampilkan gambar serta deskripsi yang sudah dibuat oleh user
- Menghapus gambar yang sudah dibuat
- Mengedit gambar yang sudah dibuat
- Mengganti foto profil
- Logout
